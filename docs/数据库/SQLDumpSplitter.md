## SQLDumpSplitter

### 简介

>
> [SQLSplitter](https://www.philiplb.de/sqlsplitter/) 是一种将任意大型 MySQL/MariaDB 转储分割成小的、可独立执行的 SQL 文件的工具。
>
> 在存在限制的情况下，它很有用。例如，像 phpMyAdmin 这样的 Web 管理工具的配置将上传限制为每个文件 2 MB。
>
> 该工具基于真实、最小、用例定制的 SQL 解析器。这使得它能够智能地拆分 SQL 文件并保留结构。
>
> 跨平台意味着它可以在 Windows、MacOS 和 Linux 上使用。

### 软件下载

```bash
wget https://sqlsplitter.com/static/downloads/SQLSplitter-1.0.0-Windows-AMD64.exe
```

### 操作

> 有三个选项，第一、选择需要拆分的文件，第二、选择拆分后保存的目录，第三、拆分成多大的文件，单位是MB。
